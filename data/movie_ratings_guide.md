# Movie Ratings and Reviews Guide

Movie ratings and reviews help estimate whether a movie is currently liked by audiences or critics. Ratings should not be the only factor in recommendations because personal mood and context matter.

Useful rating sources:
- IMDb ratings show broad audience opinions
- Rotten Tomatoes critic score shows professional reviews
- Rotten Tomatoes audience score shows viewer reactions
- Metacritic summarizes critic reviews
- Letterboxd can show film enthusiast opinions

How to use ratings:
- High audience ratings are useful for group-friendly picks
- High critic ratings are useful for quality-focused users
- Recent reviews help with new releases
- Streaming availability should be checked with web search because it changes often

If the user asks for current ratings, recent reviews, trending movies, new releases, or streaming availability, the agent should use web search.
